Game Rules

Cricket

In this 5 Cricket game 2 events Bookmaker (Odds) and Fancy (Session) it is Based on Cricket

  1. It is a 5 Over each Match.
  2. Team Scoring more run will be the winner.
  3. Difference of the wickets doesn't matter in result. If any teams 10 wickets fall before five overs that team is all out.
  4. If it is a tie (Difference of the wicket doesn't matter) bets will be cancelled.
  5. 1st batting team is given name AUS
  6. 2nd batting team is given name IND
  7. It is a 70 cards deck
    • A * 10 = 1 Run
    • 3 * 10 = 3 Run
    • 2 * 10 = 2 Run
    • 4 * 10 = 4 Run
    • 6 * 10 = 6 Run
    • 10 * 10 = 0 Run
    • K * 10 = Wicket
  8. After the First inning is finished deck will be reshuffled. In first inning the rates will be given after end of each over.
    In 2nd inning for the first 3 - overs rates will be given after end of each over, In 4th & 5th overs rates will be Given after end of each ball...

32 Cards A

  1. The deck of 32 cards there is 6,7,8,9,10,J,Q,K
  2. This is a value card game and Winning result will count on Highest Cards total.
  3. There are total 4 players, every player have default prefix points. Default points will be consider as following table.
    • Player 8: 8 points
    • Player 9: 9 points
    • Player 10: 10 points
    • Player 11: 11 points
  4. In game, every player has to count sum of default point and their own opened card's point.
  5. If in first level, the sum is same with more than one player, then that will be tie and winner tied players go for next level.
  6. This sum will go and go upto Single Player Highest Sum of Point.
  7. At last highest point card's player declare as a Winner.
